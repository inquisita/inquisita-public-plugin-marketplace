# Inquisita Plugin Marketplace

A Claude Code plugin marketplace for [Inquisita](https://inquisita.ai) — a document intelligence platform for legal, regulatory, and compliance workflows.

## Installation

Add the marketplace:

```
/plugin marketplace add inquisita/inquisita-public-marketplace
```

Browse and install plugins:

```
/plugin install inquisita-core@inquisita-marketplace
```

## Licensing

Some plugins in this marketplace are open source. The `inquisita-core` plugin is proprietary but free for Inquisita customers to use.

See individual plugin directories for specific license terms.
